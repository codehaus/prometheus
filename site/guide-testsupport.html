Tests for the termination of a Thread

There are various methods that can be used for asserting that the thread is finished:

  1. assertIsTerminated
  2. assertIsTerminatedWithoutThrowing
  3. assertIsTerminatedWithException
  4. etc

Test that a Thread is blocking

There is no way to detect of a thread is blocking within the JVM. The Thread.getState can't be trusted for providing the actual state the thread is in. The only thing that can be done is to ask the thread if it is still running by calling the assertIsStarted().

Test if exception is thrown

If you want to check if a thread has terminated, but no exception was thrown, use the following: 

    TestThread t = ...;
    t.assertTerminatedWithoutThrowing()
If you want to check if a specific exception was thrown, you can use the following: 
    TestThread t = ...;
    t.assertTerminatedWithException(RejectedExecutionException.class)
If have a reference to the exception, you can also use the following assert statement: 
        Exception ex = ....;
        TestThread t = ...;
        t.assertTerminatedWithException()
